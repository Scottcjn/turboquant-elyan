# Contributing to TurboQuant

Thank you for your interest in contributing to TurboQuant!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/your-username/turboquant-elyan.git`
3. **Create** a branch: `git checkout -b feature/your-feature`
4. **Make** your changes
5. **Test** your changes
6. **Submit** a Pull Request

## Development Setup

```bash
# Clone and setup
git clone https://github.com/your-username/turboquant-elyan.git
cd turboquant-elyan

# Install dependencies
pip install -r requirements.txt
```

## Code Style

- Follow PEP 8 for Python code
- Use meaningful names for variables and functions
- Add docstrings

## Testing

```bash
# Run tests
pytest
```
